#　各ファイルの役割
##　app.py

Flaskアプリケーション本体です。
管理画面の表示、各種APIの定義、商品登録・更新、商品削除、購入処理、万引き通知処理、AI認識結果による在庫更新を行います。

主に、Web画面とデータ管理処理をつなぐ役割を持っています。

##　data_store.py

商品情報、在庫情報、売上履歴、通知履歴、AI認識履歴を管理するファイルです。
商品登録・更新、商品削除、在庫数の変更、売上履歴の追加、通知履歴の追加、AI認識履歴の追加を行います。

現在は、プログラム内の変数を用いてデータを管理しています。

##　line_notify.py

LINE通知を送信するためのファイルです。
購入時には購入通知、万引き疑い発生時には万引き通知を送信します。

LINE Messaging APIを使用し、環境変数に設定したアクセストークンを用いて通知を行います。

##　templates/index.html

管理画面のHTMLファイルです。
商品登録・更新フォーム、現在の在庫一覧、売上履歴、通知履歴、AI認識結果履歴、動作テストボタンを表示します。

また、JavaScriptを用いてFlaskのAPIと通信し、在庫や履歴の表示を更新します。

##　static/style.css

管理画面の見た目を整えるCSSファイルです。
画面全体の背景色、カード形式のレイアウト、表、ボタン、入力フォームなどのデザインを設定しています。
