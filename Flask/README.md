各ファイルの説明
app.py

Flaskアプリケーション本体です。

管理画面の表示や、在庫情報、商品情報、売上履歴、通知履歴、AI認識履歴を取得するAPIを定義しています。

また、商品登録・更新、商品削除、購入通知、万引き通知、AI認識結果による在庫更新の処理も行います。

主な役割は以下です。

管理画面の表示
在庫情報の取得
商品情報の取得
売上履歴の取得
通知履歴の取得
AI認識履歴の取得
商品登録・更新
商品削除
購入処理
万引き通知処理
AI認識結果による在庫更新
data_store.py

商品情報、在庫情報、売上履歴、通知履歴、AI認識履歴を管理するファイルです。

現在は、プログラム内の変数を使ってデータを管理しています。

主な役割は以下です。

商品情報の管理
在庫情報の管理
売上履歴の管理
通知履歴の管理
AI認識履歴の管理
商品登録・更新
商品削除
在庫数の更新
AI認識結果による在庫数の上書き
line_notify.py

LINE Messaging APIを用いて、LINE通知を送信するためのファイルです。

購入時には購入通知、万引き疑い発生時には万引き通知を送信します。

LINE通知を利用する場合は、環境変数 LINE_CHANNEL_ACCESS_TOKEN にLINEチャネルアクセストークンを設定します。

主な役割は以下です。

LINE通知メッセージの送信
LINEチャネルアクセストークンの読み込み
通知送信結果の確認
templates/index.html

管理画面のHTMLファイルです。

商品登録・更新フォーム、現在の在庫一覧、売上履歴、通知履歴、AI認識結果履歴、動作テストボタンを表示します。

また、JavaScriptを用いてFlaskのAPIと通信し、画面の表示内容を更新します。

主な画面内容は以下です。

商品登録・更新
現在の在庫
売上履歴
通知履歴
AI認識結果履歴
動作テスト
static/style.css

管理画面の見た目を整えるCSSファイルです。

背景色、カード形式のレイアウト、表、ボタン、入力フォームなどのデザインを設定しています。

主な役割は以下です。

管理画面全体のデザイン設定
カード形式のレイアウト設定
表のデザイン設定
ボタンの色や形の設定
入力フォームのデザイン設定
